# NPM Skill

## When to use
- Managing Node.js projects
- Installing or updating dependencies
- Running npm scripts
- Publishing packages

## Best practices
- Use `package-lock.json` for reproducible builds
- Keep dependencies up to date
- Use `devDependencies` for build/test tools
- Prefer `npx` for one-off command execution

## Common commands
```bash
# Install dependencies
npm install

# Run scripts
npm run build
npm test

# Add dependency
npm install lodash
npm install -D typescript
```
